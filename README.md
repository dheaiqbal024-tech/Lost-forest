# The Lost Forest - Text Adventure Game

A simple, realistic text-based adventure game built in Python.

## Features

- **Dynamic World**: Explore multiple interconnected locations
- **Combat System**: Encounter enemies and fight them with strategy
- **Inventory System**: Collect items and use them
- **Experience System**: Gain experience points from victories
- **Player Stats**: Track health, inventory, and experience

## Game Elements

### Locations
- Forest Entrance
- Deep Forest
- Old Cabin
- Mountain Path

### Enemies
- Wolf
- Goblin
- Dark Knight
- Forest Guardian

### Items
- Rusty Key
- Healing Potion
- Ancient Map

## How to Play

Run the game:
```bash
python game.py
